# Terraform AWS EKS Infrastructure

## Overview
Enterprise-grade AWS EKS infrastructure provisioned using Terraform with modular design and remote state management.

## Key Features
- Modular Terraform architecture
- Environment isolation (dev/prod)
- Remote state with S3 & DynamoDB
- EKS cluster with managed node groups
- Cost-aware instance selection

## Why Modules?
- Reusability
- Consistency
- Scalable team collaboration

## How to Deploy
terraform init
terraform plan
terraform apply
